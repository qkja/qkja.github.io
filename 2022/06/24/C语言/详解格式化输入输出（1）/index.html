序言

这个博客就是为了更好的学习printf和scanf函数，更好的避免踩坑，不过大家要注意，一般情况很多专业的程序员会避免使用这两个函数

输出：printf()函数

printf()函数是我们经常使用的的函数之一，他的作用就是将我们想要的数据输出到指定位置，这里的指定位置包含屏幕，我们先使用一下

#include <stdio.h>

int main()
&#123;
    int a = 10;
    printf("%d\n", a);
    return 0;
&#125;

这里我们可以看到，我们使用了printf()函数

printf("%d\n",a);

这里有几点要说一下

  • % 被称为 **”转换说明”**，所谓的转换说明就是将数据按指定的格式的方法,详细的解说 :point_right:转换说明
  • %d是指将a的值以十进制的形式打印来,例如 %o :打印八进制、**%x** 打印十六进制…..
  • 这里没有指定打印的位置，可是却输出到屏幕中，实际上printf()函数默认的输出就是该窗口

prinft()错误的使用

C语言不会检测转换说明的数量和我们要输出的数据的个数是不是匹配的，但我们不小心写错的时候，程序任就会执行，这就给我们代码带来了隐患，我们一定要小心

转换说明 < 数据个数

只会从左向右打印相互匹配的数据，没有配的不打印

#include <stdio.h>
int main()
&#123;
    int a = 10;
    int b = 20;
    printf("%d", a,b);//只打印 a
    return 0;
&#125;

转换说明 > 数据个数

从左向右依次打印，没有匹配的打印随机数

#include <stdio.h>
int main()
&#123;
    int a = 10;
    int b = 20;
    printf("%d %d %d", a, b);
    return 0;
&#125;

输入：scanf()函数

和printf() 一样，scanf()也是存在特定的格式读入，这里先给出个例子，我们一起来看看

#include <stdio.h>

int main()
&#123;
    int a = 0;
    scanf("%d", &a);
    printf("%d\n", a);
    return 0;
&#125;

这些都太简单了，我们来见识一下scanf()的经常出现的问题

scanf原理

先看看下面这张图片

我总结一下

  • scanf从左向有读取数据
  • 对于每一个转换说明，scanf从输入数据中定位到合适类型数据，可以跳过空格，在不可能位置处停止这个转移字符的读取，接着下一个
  • 对于上一步骤不可能的那个字符，作为下一次读取的起始点
  • 要是中间不能读取成功，停止读取

可以跳过空白字符

可以跳过空白字符是为了找到数值的起始位置

#include <stdio.h>

int main()
&#123;
    int a = 0;
    scanf("%d", &a);
    printf("%d\n", a);
    return 0;
&#125;

停止读入

#include <stdio.h>

int main()
&#123;
    int a = 0;
    int b = 0;
    scanf("%d %d", &a,&b);
    printf("a = %d b = %d\n", a,b);
    return 0;
&#125;

读入字符串

前面我们读取的都是int型，下面以读取字符串为例

#include <stdio.h>

int main()
&#123;
    char arr[100] = &#123; 0 &#125;;
    scanf("%s", arr);
    printf("%s\n", arr);
    return 0;
&#125;

结论

scanf读入字符串时，遇见空白字符会自动停止，我们用的时候一定要小心

读入整型

我直接说过程

  1. 寻找 正负号
  2. 读取数值
  3. 遇到一个非数字的字符停止

读入浮点数

我直接说过程

  1. 寻找 正负号
  2. 读取数值（可能会包含小数点）
  3. 读取指数e或者E（可选）

例题（重要）

讲解 ：:point_right:例题讲解

你要是把这道题弄懂了，可以说是对于这个知识点完全理解了

#include <stdio.h>

int main()
&#123;
    int i = 0;
    int j = 0;
    float x = 0.0f;
    float y = 0.0f;
    scanf("%d%d%f%f", &i, &j, &x, &y); //把   1-20.3-4.0e3换行  输入进出
    printf("i = %d\n", i);
    printf("j = %d\n", j);
    printf("x = %f\n", x);
    printf("y = %f\n", y);
    return 0;
&#125;
