序言

今天主要谈谈方法的实现与调用和以及数组有关的的一些知识点，内容不多，也比较简单

方法

“方法”就是我们之前说的“函数”或者“子程序”，这里就不在一一仔细阐述。

HelloWord方法

public class TestDemo &#123;
    
    public static void myPrint() &#123;
        
        System.out.println("Hello World");
    &#125;
    
    public static void main(String[] args) &#123;
        
        myPrint();
        
    &#125;
  • myPrint函数static，因为main是static
  • 方法再main下面和上面都一样
  • 方法名采用小驼峰形式

函数重载

public class TestDemo &#123;

    public static void main(String[] args) &#123;
        
        int a = 10;
        int b = 20;
        System.out.println(sum(a,b));
        
        double c = 10.0;
        double d = 20.0;
        System.out.println(sum(c,d));
    &#125;
    
    public static int sum(int x,int y) &#123;
        
        return x + y;
    &#125;
    
    public static double sum(double x,double y) &#123;
        
        return x + y;
    &#125;
    
&#125;

这就是重载，重载要满足一些条件

  • 方法名相同
  • 方法的参数不同 (参数个数或者参数类型)
  • 方法的返回值类型不影响重载.

数组

从数组开始，Java和C语言就开始有很大的不同了，我们可以对比着学习

与C的不同

  • java：int[ ] arr;
  • C语言：int arr[ N ](注：N为一常数)

这里提一下，Java中的命名规则可以说是“形象的”，对于C语言来说
int arr1[10] 和 int arr2[20] 就不是一个数据类型的，arr1是int[10]、arr2是int[20];

一维数组

数组元素初始化

public class TestDemo &#123;
    
    public static void main(String[] args) &#123;
        
        int[] arr1 = &#123;&#125;;  //错误做法 已经分配了内存
        int[] arr2 = null;//正确做法

NULL 和 null的区别

  • NULL是0地址
  • null就是一个空对象，和0地址没有关系

三种一位数组的创建方式

public class TestDemo &#123;
    
    public static void main(String[] args) &#123;
        
        int[] arr1 = &#123;1,2,3,4,5,6&#125;;               // []内不能有数字
        int[] arr2 = new int[]&#123;1，2，3，4，5，7&#125;;   // 同样[]内不能加数字
        int[] arr3 = new int[5];                  //数组内有5个元素，每个元素默认初始化为0
    &#125;

数组元素的内存存储位置 元素名在栈区 ，元素在堆区

二维数组

如果说一维数组与C语言还有些相似的话，那么二维数组有C语言有着很大的不同，我们来仔细看看二维数组的存储

  • 数组名在栈区
  • 二维数组是一个特殊的一维数组，其每一个元素都是一个一维数组
